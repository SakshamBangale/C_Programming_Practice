## 📖 About

Welcome to **C Programming Practice**!

This repository is a structured collection of **C programming exercises**, coding challenges, interview questions, data structures, algorithms, and mini projects.

Whether you're a beginner learning C or preparing for technical interviews, this repository provides organized examples and practical exercises to strengthen your programming skills.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/SakshamBangale/c-programming-practice.git
```

Move into the project

```bash
cd c-programming-practice
```

Compile

```bash
gcc basics/01_hello_world.c -o app
```

Run

```bash
./app
```
